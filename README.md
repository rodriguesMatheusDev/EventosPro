# 📅 EventosPro

O **EventosPro** é uma plataforma completa para descoberta e gestão de eventos. O projeto foi desenvolvido para demonstrar habilidades em **Node.js**, **EJS** e **Tailwind CSS**, focando em uma experiência de usuário (UX) fluida e moderna.

---

## 🚀 Funcionalidades

- **Descoberta de Eventos:** Listagem dinâmica de eventos com filtros por categoria.
- **Sistema de Login Simulado:** Autenticação baseada em `LocalStorage`, permitindo persistência de dados no navegador sem necessidade de banco de dados externo.
- **Múltiplos Perfis:** - **Participante:** Pode se inscrever em eventos e gerenciar seus ingressos.
  - **Organizador:** Dashboard exclusiva para criação de eventos e métricas de desempenho.
- **Design Responsivo:** Interface construída com Tailwind CSS, totalmente adaptável a dispositivos móveis.
- **Feedback Visual:** Uso de SweetAlert2 para notificações e Animate.css para transições suaves.

---

## 🛠️ Tecnologias Utilizadas

- **Frontend:** EJS (Template Engine), Tailwind CSS, JavaScript (ES6+).
- **Backend:** Node.js, Express.
- **Bibliotecas:** SweetAlert2 (Modais), Animate.css (Animações), Lucide Icons/Icons8.
- **Hospedagem:** Render.

## 💻 Como rodar o projeto localmente

1. Clone o repositório:
   ```bash
   git clone [https://github.com/rodriguesMatheusDev/EventosPro.git](https://github.com/rodriguesMatheusDev/EventosPro.git)

2. Instale as dependências:
   npm install
   
3. Inicie o servidor:
   npm run dev

4. Acesse no seu navegador:
   http://localhost:3000

Atenção!

Este projeto utiliza LocalStorage para simular um banco de dados persistente no lado do cliente. Isso significa que:

Você pode criar uma conta, deslogar e logar novamente; seus dados continuarão lá.

A Dashboard de Organizador já vem populada com dados de exemplo para demonstrar como o sistema se comporta em um cenário real de uso intenso.
